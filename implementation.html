Implementation

1. Download files

Download Brofiler from their Github. You should also download the brofiler.h file.

2. Integration

Add the previous files to the project.

Include Brofiler:

#include "Brofiler.h"
#pragma comment(lib, "Brofiler/ProfilerCore32.lib")

Now, for the program to capture the frames, we must tell you which is the main loop of our application, we will use the following line at the beginning of the loop (This must be done for every thread we have):

BROFILER_FRAME("frameName")

Once indicated the main loop, we will add the following line of code at the beginning of the functions that we want to analyze:

BROFILER_CATEGORY( "functionName", Profiler::Color::colorName )

FunctionName is the name that will appear in the program and colorName the color it will use to display it in the graphics. Here is a list of all the colors that can be used:
